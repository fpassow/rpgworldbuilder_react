RPG World Builder

It's on GitHub at https://github.com/fpassow/rpgworldbuilder_react and live here.

This is a tool to help design fun, playable campaign ideas for tabletop roleplaying games. using MongoDB, Node, and React.

It's central feature is a form which is created dynamically from a JSON file (src/def.json). Users can create accounts, create campaigns, see everyone's campaigns, and clone their own copies of... read more other campaigns.

History:

  1. First built in PHP, with objects serialized to files instead of a database
  2. Changed to a single page app using jQuery because the reloads felt clunky
  3. jQuery version had Login, CampaignList, and Campaign components listening for events from each other
  4. Every component having to react correctly to every change in any other component started to hurt
  5. Rewrote with one central Big Ball of State
  6. Events go to a controller, which talks to the server and is the only thing allowed to change the Big Ball of state
  7. After any change, "view" functions used jQuery to redraw everything from the Big Ball of State
  8. Replaced jQuery-based redraw-everything with React, which achieves the same thing more efficiently
